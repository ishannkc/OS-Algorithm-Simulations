## Implemented Algorithms
- FCFS (First Come First Serve): `fcfs.py`
- SJF (Shortest Job First, Non-Preemptive): `sjf.py`
- SRTF (Shortest Remaining Time First, Preemptive SJF): `srtf.py`
- Priority (Non-Preemptive, lower number = higher priority): `priority.py`
- Round Robin (with Time Quantum set inside file): `round_robin.py`
